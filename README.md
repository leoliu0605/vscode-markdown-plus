# Markdown+

Markdown All-in-one Extension Pack for VSCode

## Features

- [ ] markdown enhanced preview with pandoc support
- [ ] markdown prettier
- [ ] excel to markdown table
- [ ] mermaid syntax highlight
- [ ] paste image support
- [ ] link checker
